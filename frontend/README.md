# Frontend

This folder will contain the static HTML, CSS, and JavaScript files for the Tender Monitor query interface.

## Status

Not yet written. Planned for the first build sprint.

## What it will contain

- `index.html` — the query interface
- `data.json` — the filtered dataset (generated by the ingestion script, not hand-edited)

## How it is hosted

GitHub Pages serves this folder as a static website. No build step, no framework, no server.
